# weather-forecast-accuracy
Historical Weather Forecast Comparison to Actuals


This project analyzes forecast accuracy using shell scripting.

## Features
- Fetches weather data using wttr.in API
- Stores historical forecasts in TSV format
- Calculates forecast accuracy
- Classifies accuracy as excellent/good/fair/poor

## Tech Used
- Bash scripting
- curl
- grep, cut, awk
- Linux CLI tools

chmod +x rx_poc.sh fc_accuracy.sh
